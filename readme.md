# Simple Counter App

Simple counter built with HTML, CSS, and JavaScript.

## Features

- Increment and decrement buttons
- Reset to zero
- Keyboard support: `+`, `-`, `r`
- Smooth hover and press effects

## Getting Started

1. Clone the repository:
`git clone https://github.com/chorobcia09/counter.git`
2. Run index.html
